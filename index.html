Documentación - Notes App

Descripción General

Esta página documenta el funcionamiento del archivo main.py correspondiente a la aplicación Notes desarrollada en Flet. La app permite crear, editar, buscar y personalizar notas con temas, tamaños de fuente y radios de borde configurables.

1. Configuración y Ajustes

El sistema carga archivos JSON para almacenar ajustes del usuario como tema, tamaño de fuente, previsualización y radio del borde. Si no existen, se generan valores por defecto. 

load_settings()
save_settings(settings)

2. Temas y Colores

La app maneja diferentes paletas visuales: Pro, Tec, Pastel, Elegante y Golden. Cada tema define colores de fondo, acentos y texto. 

get_theme_colors(theme_name)

3. Manejo de Datos

Las notas se almacenan en contenedores.json. Cada nota incluye nombre, texto, colores de fondo y opciones de previsualización. 

cargar_datos()
guardar_datos(lista)

4. Interfaz de Usuario

La UI se construye principalmente con ft.GridView y contenedores que representan notas. Incluye animaciones, gradientes, buscador, menú superior, modos de edición y vista previa.

5. Navegación

La app utiliza rutas como:

  • "/" – Pantalla principal
  • "/settings" – Ajustes
  • "/form" – Nueva nota
  • "/edit/<id>" – Edición de nota

6. Edición y Creación de Notas

La edición permite:

  • Modificar título
  • Cambiar gradiente del fondo
  • Escribir texto
  • Previsualizar formato

7. Arquitectura Interna

La aplicación sigue una estructura modular donde cada bloque cumple una función específica:

  • Funciones utilitarias: Carga y guardado de configuración y archivos JSON.
  • Clases de estilo: Definen colores, temas y variables globales usadas en la UI.
  • Funciones de datos: Manejan lectura/escritura de notas.
  • Componentes UI dinámicos: Cada nota se convierte en un contenedor con animaciones.

8. Sistema de Renderizado

Cada vez que se modifica una nota o cambia un ajuste visual, la función render_lista() reconstruye todos los contenedores de notas. Esta función garantiza que cualquier cambio se vea reflejado inmediatamente en la interfaz. 

def render_lista():
    lista_contenedores.controls.clear()
    for idx, item in enumerate(data):
        if item.get("Type", 1) == 1:
            lista_contenedores.controls.append(crear_contenedor_nota(idx, item))
    page.update()

9. Sistema de Animaciones

La aplicación utiliza animaciones de Flet como animate_scale, animate_size y BOUNCE_OUT para ofrecer una experiencia visual fluida. Las animaciones afectan elementos como:

  • Aparición/desaparición del buscador.
  • Transición de notas al mostrarse.
  • Menú lateral expandible.

10. Sistema de Búsqueda

La función de búsqueda filtra en tiempo real el contenido de todas las notas comparando el texto ingresado con el título y cuerpo de cada nota. 

def buscar_notas(e):
    texto = e.control.value.lower().strip()
    lista_contenedores.controls.clear()
    for index, item in enumerate(data):
        if texto in item.get("nombre", "").lower() or texto in item.get("text", "").lower():
            lista_contenedores.controls.append(crear_contenedor_nota(index, item))
    page.update()
